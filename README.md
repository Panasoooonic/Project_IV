# Project IV - Image Transfer System

Course project focused on transferring image data from a server to a desktop client, with progress tracking, packet handling, logging, and local file output.


## Instructions for Running the Client 
in terminal: 
1. cd ImageClient/FrameworkApp
2. dotnet build
3. dotnet run 



## Configuration
Current defaults in code:
- Output folder: `ImageClient/ImageHolder`
- Log folder: `ImageClient/Logs`
